# 📝 Simple To-Do List

A simple To-Do List app built with HTML, CSS and JavaScript.

## 🚀 Features

- Add tasks
- Delete tasks
- Prevent empty tasks
- Add tasks by pressing Enter
- Clean and simple UI

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript (Vanilla JS)

## ▶️ How to Use

1. Write a task in the input
2. Click "Add" or press Enter
3. Delete tasks with the "X" button

- style.css
- index.js
